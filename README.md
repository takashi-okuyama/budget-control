## 部門別実績のDL
### 研究費管理>研究費管理者>3 明細照会>原価センタ: 実績明細
- 原価センタグループ⇒KRBA(先端研)またはKRDB(IM研)
- 原価要素グループ⇒B3-321
- 転記日付⇒2022/04/01 to 2023/03/31 ※今年度1Qから年度末4Qまで
- 実行
### 明細表(レイアウト)
- 列:[転記日付],[伝票番号],[原価センタ],[CO 対象名],[原価要素],[原価要素名],[CO 通貨金額],[合計数量],[名称],[伝票ヘッダテキスト],[原泉先氏名/DIS備考]を選択する。
- ソート順序:[転記日付]を昇順で設定する。
- フィルタ:空
- ビュー:[AVL Grid]を選択
- 照会: [✓]チェックON
  - ⇒[水平グリッド線なし],[垂直グリッド線なし],[ソート中のマージなし],[列最適化],[エントリの上に合計行を表示]
- ファイルダウンロード(TXT保存)
- どの書式で一覧を保存しますか？⇒スプレッドシート
- Directory⇒任意
- FileName⇒「krba実績_2022.TXT」(先端研)、または「 krdb実績_2022.TXT」(IM研)で保存
- 初めての保存⇒[生成]、または上書き保存⇒[置換]
